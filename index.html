Inheritance and Prototype

Q.1 - Create 2 objects - parent and child. Attach methods to parent and use those methods in child object using parents prototype

Ans - function Parent(name) {
        this.name = name;
      }
      Parent.prototype.print = function() {
        console.log(`Hii, I am  ${this.name}`);
      }
      function Child(name) {
        Parent.call(this, name);
      }
      Child.prototype = Object.create(Parent.prototype);
      const child = new Child('Amina');
      child.print();

Here, we create a Parent object constructor which accepts a name parameter, and used the print to the Parent object's prototype. then, we create a Child object constructor, to set the name property on the Child object. after set property to Child object, we use Object, which create to inherit from the Parent object's prototype, which allows the Child object to access the print method attached to the Parent object's prototype. AtLast, there is a new instance of the Child object with the name "Amina", Print is inherited from the Parent.object's prototype, it has access to the name property on the Child object.

Q.3 - Add a method to calculate sum of all elements in Array in array's protype, use that method to calculate sum for multiple arrays

Ans - Array.prototype.sum = function() {
        return this.reduce((acc, curr) => acc + curr, 0);
      }
      
      
      const array1 = [5, 4, 3, 2, 1];
      const array2 = [50, 60, 70];
      const array3 = [500, 600, 700, 800];
      
      console.log(array1.sum()); 
      console.log(array2.sum()); 
      console.log(array3.sum());

Explanation:
Here, we used a method called sum to the Array object's prototype using the Array.prototype syntax. Then sum method which is used the reduce method to calculate the sum of all elements in the array. and then make arrays and assigned in variables named as array1, array2, and array3. We use the sum() is used to calculate the sum of all elements in each array.

Q.4 - Write a JavaScript function to retrieve all the names of object's own and inherited properties.

Ans - function allKeys(obj) {
        if (!isObject(obj)) return [];
        var keys = [];
        for (var key in obj) keys.push(key);
        return keys;
    }
    function isObject(obj) 
    {
        var type = typeof obj;
        return type === 'function' || type === 'object' && !!obj;
      }
    function Student(name) {
      this.name = name;
    }
    Student.prototype.rollno = true;
    console.log(allKeys(new Student("Amina")));

Output: [ 'name', 'rollno' ]
